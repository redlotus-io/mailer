# Development
Ensure you have the latest LTS version of Node.js installed

### Install dependencies
`yarn`

### Create env file
`cp .env.example .env`

### Setup database and api
`docker-compose up -d`

### Run db migrations
`yarn migrate:latest`

### Start server with hot reload
`yarn dev`
